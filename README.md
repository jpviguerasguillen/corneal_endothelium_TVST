# Corneal Endothelium TVST
Methods to segment cells from corneal endothelium images obtained by specular microscopy and subsequently infer the corneal parameters (cell density, coefficient of variation, and hexagonality).

Vigueras-Guillén et al. (2020) Deep learning for assessing the corneal endothelium from specular microscopy images up to one year after ultrathin-DSAEK surgery. <strong><em>Translational Vision Science & Technology (TVST)</em></strong>. (In submission)
